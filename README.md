# How to run

The notebook already has cell that install all require package or to be safe you could create new conda environment or venv the python version that's used during development is python 3.13.10

# Output

The output file is located in output folder it will contain
output file for all libraries used (TextBlob, NLTK, spaCy) which include

1. cleaned_x.txt - this file contains the cleaned text
2. words_x.txt - this file contains tokenized sentence and words
3. top10words_x.txt - this file contains top 10 words
4. time_compares.txt - this file contains table to compare the runtime of each library
